# 🏏 IPL Data Analysis Dashboard (2008–2025)

## 🧩 Overview
This project presents an interactive Power BI dashboard analyzing Indian Premier League (IPL) data from 2008 to 2025. The dashboard provides insights into team performance, player statistics, and match trends.

## 🎯 Objectives
- Analyze team-wise performance across seasons  
- Identify top-performing players  
- Track trends in runs, wickets, and match outcomes  
- Provide interactive visual insights for better understanding  

## 🛠 Tools Used
- Power BI  
- DAX (Data Analysis Expressions)  
- Excel / CSV dataset  

## 📊 Features of Dashboard
- Team performance comparison  
- Top batsmen and bowlers analysis  
- Season-wise trends  
- Match results breakdown  
- Interactive filters and slicers
